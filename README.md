# kinforma

A front-end web project focused on delivering a clean, responsive, Figma-accurate UI with modern JavaScript interactions, flexible components, and reusable utility functions.

# Overview

Kinforma is a design‑driven website built with HTML, CSS, and JavaScript. The project emphasizes:

Pixel‑perfect alignment with Figma

Clean component structure

Reusable scripts (navigation, sliders, filters)

Responsive layout for mobile → desktop

Smooth animations and transitions

# Reusable JavaScript Utilities

activeOnClick()

Adds active state to any group of clickable items.

setActiveByIndex()

Used for syncing sliders/dots.
